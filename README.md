## 🚀 Quick Start

### View Locally
Open `index.html` in your browser to see the language selector page.

### Deploy to GitHub Pages
1. Create a new GitHub repository
2. Push these files
3. Enable GitHub Pages in Settings → Pages
4. Access at: `https://YOUR_USERNAME.github.io/cosy-logistics-website/`

See `DEPLOYMENT.md` for detailed instructions.
