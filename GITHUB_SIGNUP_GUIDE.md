# 🚀 GitHub 账号注册 + 网站部署 - 完整操作指南

## ⚠️ 重要说明

由于 GitHub 账号注册需要**邮箱验证**和**安全验证**，必须你本人完成注册。

## 第 1 部分：注册 GitHub 账号（3 分钟）

### 步骤 1：打开注册页面
访问：https://github.com/signup

### 步骤 2：填写信息
1. **Email**: 输入你的邮箱（推荐用 `sales@cosylogistics.net`）
2. **Password**: 设置密码（至少 15 位，或 8 位包含数字和特殊字符）
3. **Username**: 输入用户名（例如：`cosylogistics` 或 `jack-cosy`）
   - ⚠️ **记住这个用户名！** 这是你的网站地址的一部分
4. 点击 **"Verify"** 完成验证

### 步骤 3：邮箱验证
1. 登录你的邮箱
2. 找到 GitHub 的验证邮件
3. 输入验证码

### 步骤 4：完成注册
回答几个简单问题（可以随便选），完成注册！
